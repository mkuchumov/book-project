Каталог книг
------------


### Запуск через Docker compose

~~~
docker compose up -d --build
~~~

### Запуск командной строки:
~~~
docker exec -it yii_php bash
~~~

### Установка пакетов
~~~
composer install
~~~

### Запуск миграций:
~~~
cd book-project
php yii migrate
~~~

### Проект доступен по адресу
~~~
http://localhost/
~~~
